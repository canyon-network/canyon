pub struct DepthInfo<BlockNumber> {
    pub blocks: BlockNumber,
    pub total_depth: Depth,
}
Expand description

Historical info about the average value of depth.

This struct is used for calculating the historical average depth of a validator, which implies the storage capacity per validator.

Fields

blocks: BlockNumber

Number of blocks authored by a validator since the weave is non-empty.

The blocks here is not equal to the number of total blocks authored by a validator from the genesis block because the poa contruction is skipped when the weave is empty, the blocks authored in that period are not counted.

total_depth: Depth

Sum of all depths so far.

pub fn as_storage_capacity(&self) -> Permill

Returns the calculated storage capacity.

In theory, the greater the historical average depth, the less the storage of node stored locally.

   average_depth = self.total_depth / self.blocks

storage_capacity = 1 / average_depth
                 = self.blocks / self.total_depth
